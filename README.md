> **Transform raw data into professional, AI-powered insights in seconds — no coding required!**

An intelligent, autonomous system that analyzes your data and generates comprehensive reports with interactive charts, strategic business recommendations, and natural language narratives. Built with a modern React (Vite) frontend and a Python FastAPI backend powered by LangChain and Groq LLMs.

---

## ✨ Features

🚀 **Fully Autonomous** - Upload data (.csv or .xlsx) → Get immediate insights.
🤖 **AI-Powered Narratives** - Deeply detailed storytelling reports and actionable business recommendations generated by advanced LLMs (Llama 3.3 via Groq/LangChain).
📊 **Interactive Visualizations** - Beautiful, dynamic charts rendered with Plotly (embedded React wrappers) spanning distributions, correlations, and trends.
🌐 **Premium Web Interface** - A highly polished, responsive dashboard built with React, Tailwind CSS, Framer Motion, and Lucide React icons.
🐍 **Robust Backend API** - A clean FastAPI architecture handling session management, data cleaning (Pandas), and prompt orchestration.
🔍 **Data Chat** - Talk to your dataset directly using an AI Data Agent.

---

## 🏗️ Architecture

The project is split into a decoupled modern full-stack architecture:

### 🖥️ Frontend (`/frontend`)
- **Framework:** React + Vite
- **Styling:** Tailwind CSS + Glassmorphism design patterns
- **Animations:** Framer Motion
- **Visualizations:** React-Plotly.js + Recharts (KPIs)
- **Routing:** React Router DOM (Dynamic Dashboard, Charts, Storytelling Report, Data Chat)

### ⚙️ Backend (`/backend`)
- **Framework:** FastAPI (Python)
- **Data Processing:** Pandas (cleaning, imputing, typing)
- **AI Core:** LangChain + Groq API (`llama-3.3-70b-versatile`)
- **Endpoints:** File Upload, Dataset Summaries, Chart Generation, AI Storytelling, and Data Q&A.

## 🚀 Quick Start

### 1. Backend Setup

```bash
cd backend

# Option: Create a virtual environment
python -m venv .venv
source .venv/Scripts/activate  # Windows
# source .venv/bin/activate    # Mac/Linux

# Install dependencies
pip install -r requirements.txt

# Environment Variables
# Create a .env file in the backend directory
# Add your Groq API key:
# XAI_API_KEY=gsk_your_api_key_here

# Launch the FastAPI Server
uvicorn main:app --reload --port 8000
```
*The API will be available at `http://localhost:8000`. You can view the swagger docs at `http://localhost:8000/docs`.*

### 2. Frontend Setup

Open a new terminal:

```bash
cd frontend

# Install Node dependencies
npm install

# Launch the React development server
npm run dev
```
*The web interface will typically be available at `http://localhost:5173`.*

## 🎨 User Interface Guide

1. **Gate / Login:** The app starts securely behind a mockup authentication wall.
2. **Dashboard:** Once logged in, use the floating **New Analysis** button in the Top Navbar to upload your dataset.
3. **Dataset Summary:** View the raw data schema, missing values count, and a paginated table of your records.
4. **Interactive Charts:** Browse AI-selected charts based on your data types. Click to zoom in on full-screen modals accompanied by "Beginner's Guide" AI insights.
5. **Storytelling Report:** Read a deeply generated executive summary, key insights, trends, and anomalies tailored strictly to your data numbers.
6. **Recommendations:** Review high-priority strategic actions and click "Take Action" to log your decisions.
7. **AI Chat:** Ask direct ad-hoc questions about your dataset.
